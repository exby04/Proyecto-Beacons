# Proyecto Beacons

## Estructura
- android-app → aplicación Android receptora de beacons.
- arduino → código Arduino emisor de beacon (iBeacon).
- server → lógica de servidor para recibir datos (Node/PHP/Firebase).
- database → scripts SQL para la base de datos.
- docs → documentación y PDFs del proyecto.

## Cómo correr
1. Cargar el código en el Arduino (usa placa nRF52).
2. Abrir la app Android y pulsar "Buscar dispositivos".
3. Ver los beacons detectados en Logcat.
